Engineering Mathematics-II

Regular Exam
▼

Unit 1: Differential Equations of First Order

  • Linear DE: dy/dx + Py = Q, solved using integrating factor templates.
I.F. = e^(∫P dx) ==> Solution: y * I.F. = ∫(Q * I.F.)dx + C
  • Bernoulli's Equation: Reduces non-linear models dy/dx + Py = Q * y^n via substitution v = y^(1-n).
  • Exact Conditions: Equation Mdx + Ndy = 0 is exact if ∂M/∂y = ∂N/∂x.

Unit 2: Linear DE of Higher Order

  • Operator Forms: f(D)y = Q(x) via Complete Solutions: y = C.F. + P.I.
Type 1 [e^(ax)]: Set D = a
Type 2 [sin(ax)]: Set D^2 = -(a^2)
