Unit 1: First-Order Ordinary Differential Equations

  • Bernoulli Differential Equations: Non-linear reductions via substitution down to linear variants.
  • Exact DEs: Applying the partial validation check ($\frac{\partial M}{\partial y} = \frac{\partial N}{\partial x}$) and execution tracking.
  • Physical Modeling: Newton's Law of Cooling, orthogonal trajectories computation, and rate models.
Standard Integrating Factor:
I.F. = e^(∫P dx) ==> General Solution: y * I.F. = ∫(Q * I.F.)dx + C

Unit 2: Higher-Order Linear Differential Equations

  • Complementary Functions (C.F.): Linear constant coefficient solutions via auxiliary equation root types.
  • Particular Integrals Shortcuts: Quick operators for driving parameters shaped as $e^{ax}$, $\sin(ax)$, $\cos(ax)$, $x^m$, or $e^{ax}V(x)$.
  • Variation of Parameters: Finding non-homogeneous solutions using Wronskian determinant systems.
  • Transient Networks Applications: Setting up equations for LCR circuit loops under transient changes.
Wronskian Determinant Verification:
W = |y1 y2; y1' y2'|

Unit 3: Partial Differential Equations & Applications

  • PDE Formulations: Eliminating arbitrary constants and functional boundaries to frame surface models.
  • Lagrange's Linear Equations: Systematically solving first-order linear expressions ($Pp + Qq = R$).
  • Multivariable Optimization: Identifying local minima, maxima, and saddle points using the discriminant value check ($rt - s^2$).

Unit 4: Vector Differentiation

  • Vector Kinematics: Velocity and acceleration derivation along space curves via position vectors.
  • Gradient Operators: Calculating normal vectors to level surfaces and finding Directional Derivatives.
  • Divergence & Curl metrics: Evaluating outward flux vectors ($\nabla \cdot \mathbf{F}$) and rotational vorticity properties ($\nabla \times \mathbf{F}$).
  • Vector Fields Classification: Solenoidal criteria proofs and irrotational field potential determinations.

Unit 5: Vector Integration Theorems

  • Path & Flux Integrals: Work done loop math, surface configuration flux tracking, and line integration steps.
  • Green's Theorem: Changing closed line integrals across a planar region into simple 2D double integrals.
  • Stokes' Theorem: Swapping line boundaries integration path with surface integration of the vector field's curl.
  • Gauss Divergence Theorem: Evaluating net outward surface flux via direct 3D volume integration.
Gauss Divergence Transformation:
∯ F · n dS = ∭ (∇ · F) dV

E = E0 - (0.0591 / n) * log10([Products] / [Reactants])
