Engineering Mathematics-II

Regular Exam
▼

Unit 1: Differential Equations of First Order and First Degree

  • Linear Differential Equations: Formulated as dy/dx + Py = Q, solved using integrating factor templates.
I.F. = e^(∫P dx) ==> Solution: y * I.F. = ∫(Q * I.F.)dx + C
  • Bernoulli's Equation: Reduces non-linear models of the form dy/dx + Py = Q * y^n by making substitution v = y^(1-n).
  • Exact Conditions: Equation Mdx + Ndy = 0 is exact if and only if ∂M/∂y = ∂N/∂x. Non-exact variations require verification of structural Integrating Factors.
  • Applications: Newton's Law of Cooling (dT/dt = -k(T - Ts)), radioactive decay pathways, and standard LR / RC electric current growth loops.

Unit 2: Linear Differential Equations of Higher Order

  • Operator Formulations: Solves higher-order equations f(D)y = Q(x) via Complete Solutions: y = C.F. + P.I.
  • Particular Integral (P.I.) Operational Shortcuts:
Type 1 [Q(x) = e^(ax)]: Set D = a, provided f(a) != 0.
Type 2 [Q(x) = sin(ax) or cos(ax)]: Set D^2 = -(a^2), provided f(-(a^2)) != 0.
Type 3 [Q(x) = x^m]: Expand [f(D)]^(-1) using algebraic binomial paths.
  • Advanced Reduction Paths: Method of Variation of Parameters using Wronskian determinants. Cauchy-Euler systems resolved by mapping x = e^z.

Unit 3: Partial Differential Equations (PDE)

  • Formation Schemes: Deriving partial differential surfaces by systematically removing arbitrary constants or breaking functional loops.
  • Lagrange's First Order Method: Formulates solutions for Pp + Qq = R expressions using simultaneous ratios.
Auxiliary Equation System: dx / P = dy / Q = dz / R

Unit 4: Vector Differentiation

  • Vector Operations: Applying the Del operator (∇ = i∂/∂x + j∂/∂y + k∂/∂z) to fields.
  • Gradient & Directional Derivatives: ∇Φ identifies normal vectors to surfaces. Max rates of change derived via directional dot products.
  • Divergence & Curl Fields: Divergence (∇ · F) maps local volumetric expansion. Curl (∇ x F) tracks local axis rotation speeds.
Solenoidal Condition: ∇ · F = 0 || Irrotational Condition: ∇ x F = 0

Unit 5: Vector Integration & Field Theorems

  • Integration Metrics: Line Integrals track Work Done (∫F·dr). Surface integrals assess Flux streams.
  • The Core Integral Proof Structures:
Green's Theorem: ∫(Mdx + Ndy) = ∫∫(∂N/∂x - ∂M/∂y)dxdy [2D Plane Area Conversion]
Stoke's Theorem: ∫F · dr = ∫∫(∇ x F) · n dS [3D Line to Surface Curl Conversion]
Gauss Divergence: ∫∫F · n dS = ∫∫∫(∇ · F) dV [Surface Envelopes to Volumetric Integral Conversion]

Network Analysis

Core ECE Track
▼

Unit 1: DC Circuits & Network Theorems

  • Analysis Loops: Nodal analysis driven by KCL setups. Mesh equations built using structural loop matrices.
  • Dependent Sources: Managing Voltage and Current controlled elements during calculations.
  • Transform Theorems: Thevenin values (Vth, Rth) and Norton targets (In, Rn). Max Power occurs when load resistance matching satisfies:
Pure DC Loop: RL = Rth || Complex AC Loop: ZL = Zth* (Complex Conjugate Match)

Unit 2: Transients & Laplace domain

  • Time Domain Transitions: Managing state changes in R-L and R-C circuits. Inductors maintain continuous currents i(0-)=i(0+). Capacitors block sudden shifts in voltage v(0-)=v(0+).
  • s-Domain Equivalents: Converting components using Laplace Transforms.
Inductors: sL impedance with series initial voltage source [L*i(0-)]
Capacitors: 1/sC impedance with series initial voltage source [v(0-)/s]

Unit 3: Steady State AC Networks

  • Sinusoidal Tracking: Working with reactive impedances (Z = R + jX) and reactive admitances (Y = G + jB).
  • Phasor Architectures: Performing vector shifts and angular arithmetic on complex waveforms using Euler rules.
  • Star-Delta Interconversions: Simplifies complex multi-terminal networks.
Delta to Star: R1 = (Rb * Rc) / (Ra + Rb + Rc)

Unit 4: Resonance & Coupled Loops

  • Resonance Parameters: Conditions where inductive reactances equal capacitive components, driving input currents entirely in phase with supply lines.
Resonant Frequency: ω0 = 1 / sqrt(L * C) || Quality Factor: Q = ω0*L / R = 1 / (ω0*C*R)
  • Coupled Magnetic Systems: Mutual inductance (M) calculations combined with dot rules to determine if induced voltages add or subtract.

    • Unit 5: Two-Port Networks

    • Parameter Configurations: Solving for matrix indices using black box port boundaries.
    Z-Parameters (Open Circuit): V1 = Z11*I1 + Z12*I2 || V2 = Z21*I1 + Z22*I2
    Y-Parameters (Short Circuit): I1 = Y11*V1 + Y12*V2 || I2 = Y21*V1 + Y22*V2
    ABCD Parameters (Transmission): V1 = A*V2 - B*I2 || I1 = C*V2 - D*I2 [Ideal for Cascaded Links]

    Data Structures

    Computing Module
    ▼

    Unit 1: Arrays & Linked Lists

    • Complexity Boundaries: Tracking runtime constraints using Big-O notation metrics. Accessing static arrays requires O(1) steps. Searching unordered lists takes O(n) worst-case processing cycles.
    • Dynamic Nodes Implementations: Memory management via pointers.
    Singly Linked Node: [Data | Next Pointer]
    Doubly Linked Node: [Prev Pointer | Data | Next Pointer] -> Supports bidirectional steps
    Circular Linked Node: Last node's Next pointer references head position -> Loop tracking

    Unit 2: Stacks & Queues

    • Stack Structures (LIFO Pipeline): All insertions and deletions take place at a single 'Top' index point. Handled via Push and Pop methods. Supports compiler tracking loops, bracket parity tests, and prefix conversions.
    • Queue Structures (FIFO Pipeline): Element additions occur at the Rear boundary, and removals occur at Front boundary lines.
    Circular Queue Condition: Rear = (Rear + 1) % Max_Size [Resolves data space leaks]

    Unit 3: Searching & Sorting Techniques

    • Search Complexities: Linear scanning loops execute in O(n) times. Binary configurations require pre-sorted inputs and partition scopes in O(log n) steps.
    • Sorting Engine Algorithms:
    Bubble / Selection / Insertion Sort: Average and worst case processing metrics at O(n^2).
    Quick Sort: Partitioning arrays around dynamically selected pivot indices. O(n log n) average step counts.
    Merge Sort: Divide-and-conquer strategy that continuously breaks arrays into halves before recombining. Guarantees O(n log n).

    Unit 4: Binary Trees & BST

    • Tree Architectures: Non-linear node systems tracing roots and child leaf targets.
    • Binary Search Trees (BST): Left child node values must be less than parent node values. Right child node values must be greater than parent node values.
    • Recursive Traversals Execution Paths:
    Pre-Order: Visit Root -> Process Left Subtree -> Process Right Subtree
    In-Order: Process Left Subtree -> Visit Root -> Process Right Subtree [Produces sorted list for BST]
    Post-Order: Process Left Subtree -> Process Right Subtree -> Visit Root

    Unit 5: Graphs Structural Methods

    • Network Storage Matrices: Storing connections using 2D binary Adjacency Arrays or memory-efficient Adjacency Link lists.
    • Traversal Engines:
    Breadth-First Search (BFS): Explores tracking networks layer by layer. Powered by a Queue array.
    Depth-First Search (DFS): Traverses deeply along each branch before backtracking. Powered by recursive stacks.

    Engineering Chemistry

    Applied Science
    ▼

    Unit 1: Atomic & Molecular Properties

    • Molecular Orbital Theory (MOT): Linear combinations of atomic orbitals form discrete bonding and anti-bonding molecular energy states.
    • Electronic Config Maps: Drawing explicit energy level diagrams for molecules like O2 and CO.
    Bond Order Calculation: BO = (No. of Bonding Electrons - No. of Anti-bonding Electrons) / 2
    • Magnetic Categorization: Unpaired electrons flag paramagnetic trends. Complete paired allocations create diamagnetic shields.

    Unit 2: Solid Semiconductors Chemistry

    • Solid Energy States: Valence bands full of electrons sit beneath an energy gap, above which lies the empty conduction band.
    • Doping Injections: Modifying material conductivity.
    n-Type Semiconductors: Silicon doped with Pentavalent targets (Phosphorus). Extra valence electrons become majority carriers.
    p-Type Semiconductors: Silicon doped with Trivalent targets (Boron). Electron absences (Holes) become majority carriers.

    Unit 3: Electrochemistry & Battery cells

    • Galvanic Cell Interfaces: Anode tracking sites undergo oxidation reactions. Cathode boundaries manage reduction pathways.
    • Thermodynamic Potentials: Cell potential tracking calculations are computed using ion concentrations and the Nernst Equation:
    E_Cell = E0_Cell - (0.0591 / n) * log10([Anode Products Ions] / [Cathode Reactant Ions])
    • Energy Packs Configurations: Discharging and charging chemistry lines behind industrial Lead-Acid arrays, commercial Lithium-ion battery packs, and clean Hydrogen-Oxygen fuel cells.

    Unit 4: Polymers & Elastomers

    • Synthesis Vectors: Chain growth additions versus step-growth condensation processes.
    • Thermal Variations: Thermoplastics exhibit linear structures that soften easily when heated. Thermosetting polymers rely on strong cross-linked networks that set permanently.
    • Synthetic Formulations: Molecular structural layouts and preparation steps for Nylon-6,6 threads, Terylene structures, and synthetic Buna-S and Buna-N rubbers.

    Unit 5: Spectroscopy Instrumental Blueprints

    • Photometric Laws: Spectroscopic analysis runs entirely on concentration scaling paths.
    Beer-Lambert Formulation: Absorbance (A) = log10(I0 / I) = ε * c * l [ε = Absorptivity, c = Concentration, l = Path Length]
    • Analytical Instruments Architectures: Optomechanical component sequences and layout paths for UV-Visible light detectors, FTIR systems mapping molecule vibrations, and HPLC separation tubes.

    1-1 Linear Algebra & Calculus

    Supply Target
    ▼

    Matrix Architecture & Consistency Systems

    • Rank Evaluations: Reducing matrices to Row Echelon Form using systemic elementary row transformations. The number of non-zero rows defines the Rank, ρ(A).
    • Consistency Verifications: Checking non-homogeneous systems AX = B with the Augmented Matrix [A|B].
    Consistent System: Rank(A) = Rank([A|B]) [If Rank = No. of variables n -> Unique Solution]
    Infinite Solutions: Rank(A) = Rank([A|B]) & Rank < n [Requires parameter assignments]
    Inconsistent System: Rank(A) != Rank([A|B]) [Zero Solution Space]

    Eigenvalues & Linear Transformations

    • Characteristic Tracking: Solving for values of λ that satisfy the characteristic determinant equation: |A - λI| = 0.
    • Eigenvector Inferences: Substituting derived values of λ back into (A - λI)X = 0 to isolate corresponding non-trivial directional vector baselines.
    • Cayley-Hamilton Theorem Application: Every square matrix satisfies its own characteristic equation, providing a streamlined way to calculate higher matrix exponents and inverse configurations:
    Example Matrix Polynomial: A^2 - 5A + 7I = 0 ==> Inverse Form: A^-1 = (1/7) * (5I - A)

    Single Variable Differential Calculus

    • Mean Value Theorems Formulations:
    Rolle's Theorem: If f(a)=f(b) on continuous/differentiable intervals, then there exists a point c where f'(c) = 0.
    Lagrange's MVT: There exists a point c where f'(c) = [f(b) - f(a)] / (b - a).
    Cauchy's MVT: There exists a point c where f'(c) / g'(c) = [f(b) - f(a)] / [g(b) - g(a)].
    • Indeterminate Limit Forms: Resolving complex expressions of type (0/0) or (∞/∞) via L'Hôpital's Rule by calculating derivatives of the numerator and denominator independently until the limit evaluates cleanly.
